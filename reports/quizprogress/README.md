# gimidashboardreports_quizprogress

Quiz progress report for `local_gimidashboard`.

## What it shows

- available quizzes
- quizzes attempted
- total attempts
- submission rate
- learner drill-down
- top wrong questions
- cohort filter

## Install

Copy this folder to:

`local/gimidashboard/reports/quizprogress`

Then visit Moodle notifications or purge caches.
